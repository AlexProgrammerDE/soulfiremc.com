## Top-level navigation

After you connect, the left sidebar is split into personal navigation and, if you have permission, server administration.

### User section

| Area | Use it for |
| --- | --- |
| **Instances** | Create instances and open existing ones |
| **Access** | Manage access tokens and user-scoped access |
| **Settings** | Change personal client settings |

### Admin section

You only see these entries when your user has admin access on the connected server.

| Area | Use it for |
| --- | --- |
| **Overview** | See server-wide status |
| **Logs** | Inspect server logs |
| **Terminal** | Run server-scoped commands |
| **Users** | Create users and manage permissions |
| **Scripts** | Manage server-scoped scripts when your setup uses them |
| **Server Settings** | Change server configuration |
| **Dev Settings** | Adjust advanced developer options |

## Inside an instance

Most day-to-day work happens inside an instance.

### Controls

| Area | Use it for |
| --- | --- |
| **Overview** | Start, pause, stop, and inspect the session |
| **Terminal** | Run commands against that instance |
| **Audit Log** | Review actions taken on the instance |

### Settings

| Area | Use it for |
| --- | --- |
| **Bot Settings** | Set server address, bot amount, version, join delays, and timeouts |
| **Account Settings** | Import accounts and control how they are used |
| **Proxy Settings** | Import proxies and set proxy behavior |
| **AI Settings** | Configure AI-backed features |
| **Automation Settings** | Tune native automation collaboration, recovery, and safety behavior |
| **Instance Settings** | Edit instance-specific metadata |

### Automation and extensions

| Area | Use it for |
| --- | --- |
| **Plugins** | Enable or disable built-in plugins |
| **Plugin Settings** | Configure enabled plugins |
| **Scripts** | Build and manage visual automation for the instance |

## Personal preferences

- Use the theme toggle in the bottom-left corner to switch between light and dark mode.
- Use the language selector next to it to change the UI language when translations are available.
