Use this section if you are new to SoulFire or if you want the shortest path to a working setup.
If you get stuck, ask in our [Discord server](https://soulfiremc.com/discord).

<Callout type="info">
  Most users should start with `Integrated server (local)` in the GUI.
  It is the simplest path and does not require a separate backend.
</Callout>
