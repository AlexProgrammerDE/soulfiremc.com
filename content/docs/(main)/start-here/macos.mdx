## Basic Installation

Here's the quickest way to install SoulFire on your macOS PC.

<Steps>
  <Step>
    ### Download the software
    Visit the [download page](/download) and select macOS with the correct CPU architecture to grab the latest `.dmg`.
  </Step>

  <Step>
    ### Run the installer
    Click your downloads list in the toolbar. Hold down the `Control` key while clicking the installer.
    This will open a menu. Click "Open" to run the installer.

    If you skip the `Control` key, macOS will block the installer. That's part of its security model.
  </Step>

  <Step>
    ### Ignore warnings
    You may receive a warning that the installer is from an unidentified developer.
    You can continue with the installation.
  </Step>

  <Step>
    ### Follow the installer
    Follow the installer's instructions and accept the license. You can leave most settings at their default values.
  </Step>

  <Step>
    ### That's it!
    SoulFire should now be installed on your macOS PC. Open it from your Applications folder.
  </Step>
</Steps>
