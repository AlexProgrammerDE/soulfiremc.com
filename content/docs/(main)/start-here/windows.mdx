## Basic Installation

Here's the quickest way to install SoulFire on your Windows PC.

<Steps>
  <Step>
    ### Download the software
    Visit the [download page](/download) and choose Windows with your CPU architecture to grab the latest `.exe` installer.
  </Step>

  <Step>
    ### Run the installer
    Go into the folder where you downloaded the installer and double-click it.
  </Step>

  <Step>
    ### Ignore warnings
    You may get a warning from Windows Defender or your browser. You can ignore it and continue.
    For Windows Defender, click "More info" and then "Run anyway".

    This is common with newer software that has not built up much download history yet.
  </Step>

  <Step>
    ### Follow the installer
    Follow the installer's instructions. You can leave most settings at their default values.
  </Step>

  <Step>
    ### That's it!
    SoulFire should now be installed on your Windows PC. Open it from the Start menu, the desktop icon,
    or Windows search.
  </Step>
</Steps>
